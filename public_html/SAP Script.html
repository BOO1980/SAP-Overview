SAP Script

  • SAP Script is essentially a tool for the presentation of business data
  • Business data is presented in a SAP environment using mostly reporting tools
    • Write Statement
    • ALV (ABAP List Viewer
    • ALV Classes
    • ALV Functionalities
  • SAP Script provides a special case of business data presentation.
  • In business applications, data has to be presented in modes
  • A specific type of business document layout might require separate fort types and font sizes to be used in different parts of the business document
  • ABAP environment reporting tools do not provide all the facilities, hence why we have SAP script which is a tool to create and maintain business document layouts and produce business documents
  • SAP script is used to formate and generate business document types like:
    • purchase requisitions
    • requests for quotations
    • purchase orders
    • sales orders
    • delivery notes
    • invoices
    • credit memos
    • debit memos
    • invoices cancellations
  • SAP script can be used to generate a list of reports
